# Optimal Portfolio Allocation using CAPM

## 1. Problem Description
이 프로젝트는 CAPM 이론을 기반으로,
주어진 자산 데이터에서 위험 대비 수익이 최대가 되는
포트폴리오를 수치적으로 구성하는 것을 목표로 한다.

## 2. Mathematical Background
- Expected return
- Variance and covariance
- Sharpe ratio
본 프로젝트에서는 자산 수익률의 공분산 행렬을 이용하여
포트폴리오 위험을 계산하였다.

## 3. Methodology
1. 자산 수익률 데이터 전처리
2. 공분산 행렬 계산
3. 무작위 포트폴리오 시뮬레이션
4. 샤프 비율 최대화 포트폴리오 선택

## 4. Results
시뮬레이션 결과, 특정 위험 수준에서
가장 높은 샤프 비율을 가지는 포트폴리오를 확인할 수 있었다.

## 5. Interpretation
이 결과는 CAPM 가정 하에서
위험 대비 기대 수익을 최대화하는 자산 조합이
존재함을 보여준다.

## 6. Limitations & Future Work
- 정규분포 가정의 한계
- 향후 확률 모형을 적용한 확장 가능성
